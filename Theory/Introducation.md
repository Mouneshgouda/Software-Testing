# Software testing
- Software testing is the process of evaluating a program to find bugs, verify functionality, and ensure it meets requirements before delivery to users

## Why Software testing
- Software testing is absolutely essential because developers are human, complex code naturally contains defects,
  and releasing untested code can lead to massive financial loss, data breaches, or complete system failures

- According to data compiled by the IBM System Sciences Institute, fixing a bug after a product launches costs 15 times more than fixing 
  it during implementation and 100 times more than fixing it during the initial design phase
